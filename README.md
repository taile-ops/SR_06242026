# Skin Restoration Aesthetics Production Site

Static website package ready for GitHub Pages testing and Hostinger deployment.

## Included
- Multi-page website
- Dedicated services pages
- Dedicated treatment pages
- Shop disabled with Coming Soon badge
- Announcement bar
- SEO metadata
- Sitemap and robots.txt
- Manifest and SVG icon set
- CSS and JavaScript assets
- Legal placeholder pages
- GitHub Pages CNAME file

## GitHub Pages Testing
1. Create a new GitHub repository.
2. Upload every file and folder from this package to the repository root.
3. Go to Settings -> Pages.
4. Source: Deploy from branch.
5. Branch: main, folder: /root.
6. Open the GitHub Pages URL.

## Hostinger Deployment
Upload all files and folders into public_html.

## Before Final Launch
Replace placeholder contact details, business address, legal policy text, booking URL, and real treatment/pricing details.


## Verification Notes
- This verified package uses relative links so it works from a GitHub Pages project URL like `https://username.github.io/repository/`.
- The included `CNAME` file points to `skinrestoration.cloud`. Remove `CNAME` if you only want temporary GitHub repository testing without connecting the custom domain.
